* Request URL https://en.wikipedia.org/wiki/HTTP
* Request Method - **GET**
* Status Code - **304 Not Modified**
* Remote Address - `185.15.59.224:443`
* Referrer Policy - strict-origin-when-cross-origin

Метод **GET** используется, потому что он предназначен для получения данных пользователем.

**304 Not Modified** -- страница взята из кэша, если страницу загружаем впервые, будет статус 200 (ОК)

**То есть:**
- сервер НЕ отправляет HTML заново
- браузер берёт страницу из своего кэша
- экономится трафик и время загрузки
